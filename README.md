# Pool Builder

An interactive pool builder, inspired by [Planet Coaster 2](https://en.wikipedia.org/wiki/Planet_Coaster_2). [Live demo](https://jj-143.github.io/pool-builder)

## Development

```bash
# Install packages
npm install

# Run dev server
npm run dev
```
