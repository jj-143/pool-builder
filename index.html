Pool Builder

Inspired by Planet Coaster 2. Source code


Features

  • Customizable pool geometry
  • Directional light (the gizmo over the pool as sun)
  • Heightfield water simulation
  • Caustics, reflections & refractions
  • [Normal mode] Drag water with Mouse Left

Controls

  • Mouse Left Rotate
  • Mouse Right Pan
  • Mouse Wheel Zoom in/out

Edit mode

  • Mouse Left on the Node to select and use the gizmo
  • Mouse Right on the Node to remove it
  • Mouse Left on the Wall to split in half

Shortcuts

  • Tab Toggle Normal/Edit mode
  • Space Play/Stop simulation
  • 1 Draw rectangle
  • 2 Draw square
  • 3 Draw circle
  • 4 Draw plus
  • 5 Free draw (counter-clock-wise)
  • Ctrl+C Clear
  • Alt+Shift+Z Toggle overlays
